# 未来驰新能源汽车销售网站

## 项目说明

这是一个现代化的新能源汽车销售网站，展示未来驰 X1 概念车。

## 文件结构

- `index.html` - 主页面
- `styles.css` - 样式文件
- `script.js` - 交互脚本
- `concept-car.jpg` - 概念车图片（需要添加）

## 概念车图片说明

请将您的概念车图片命名为 `concept-car.jpg` 并放在 `ev-website` 文件夹中。

**建议图片规格：**
- 分辨率：1920x1080 或更高
- 格式：JPG 或 PNG
- 建议展示角度：车辆侧面45度角，突出流线型设计

**如果暂时没有图片，可以使用以下方式：**

1. 使用 AI 图片生成工具（如 Midjourney、DALL-E、Stable Diffusion）生成概念车图片
2. 使用占位图片服务（临时）
3. 使用渐变色背景作为临时替代

## 如何运行

1. 在浏览器中直接打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve
   ```

## 功能特点

- 响应式设计，支持移动端
- 平滑滚动导航
- 动画效果
- 预约试驾表单
- 车型参数展示
- 核心特性介绍

## 自定义

您可以在 `index.html` 中修改：
- 车型名称和参数
- 公司名称
- 联系信息
- 特性描述

在 `styles.css` 中修改：
- 配色方案
- 字体样式
- 布局设计
